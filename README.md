# Home Finance API

API para la gestión de finanzas domésticas que utiliza **IA** para proporcionar consejos financieros en tiempo real.

## Key Features
- Gestión de gastos privados y compartidos.
- Integración con Google Gemini para análisis de consumo y asesoría financiera.

## Stack
- Java 21, Spring Boot, Maven.
- PostgreSQL (Supabase).
- Spring Security + JSON Web Tokens.
- Spring AI + Google Gemini Api.
- Docker.
- React Native.
